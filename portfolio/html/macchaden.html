WORKS

抹茶伝（老舗抹茶屋の架空サイト）

サイトの概要

京都祇園に本店を構える老舗抹茶屋のWebサイトを制作しました。
地元が京都で抹茶が好きなので、抹茶の魅力が伝わる作品を制作したいと思い、制作しました。

ターゲット

抹茶好きの20〜30代の男女

目的

お店の雰囲気を知ってもらい、来店者を増やす。

制作ポイント

  • 明朝体、縦書きのタイトル、和柄を取り入れることで和を演出
  • 商品の鮮やかな緑を引き立たせるため、配色はモノトーンを基調に
  • 店舗のイメージがわかるように、カルーセルで内装の写真を表示（slick.js）
  • お知らせはスクロールを使用することでコンパクトに

制作期間

5日間

使用ツール

  • Photoshop
  • Visual studio code
