WORKS

Omomuki（ゲストハウスの架空サイト）

サイトの概要

世界中の旅行者を迎え入れるゲストハウスのWebサイト。
オーナーが温かく迎え入れてくれたり、旅行者同士と気軽に話せたり、そんなゲストハウスが好きです。ゲストハウスの良さを広めたいと考え制作に至りました。

ターゲット

  • 国内外の東京への観光客
  • 旅行者同士でコミュニケーションをとりたい人
  • 英語を話したい日本人

目的

  • 旅行者の集客
  • ゲストハウスの紹介

制作ポイント

  • ファーストビューに予約フォームを配置
  • 予約フォームのチェックインの初期値は今日に
  • 固定サイドナビゲーション
  • ゲストハウスの雰囲気が分かりやすいよう写真を多く使用

制作期間

6日間

使用ツール

  • Photoshop
  • Dreamweaver
  • Visual studio code
